paggawa ng vite react app
- npm create vite@latest nameNgFolder --template react  // in our case frontend ang name kase frontend ito
- cd frontend  // cd means change directory, ginagamit to para makalioat ng folder gamit command line interface (CLI, yung terminal)
- npm install  // ginagamit para makapag install ng mga dependencies ng project

// after installing vite at react libraries naman na gagamitin 

- npm install axios react-router-dom
// si axios ginagamit para makapagperform tayo ng CRUD functionality
// si react-router-dom ginagamit para makapagroute ng mga pages sa ating app

- npm run dev  // ginagamit para makapag start ng project
